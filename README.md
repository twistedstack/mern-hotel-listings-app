# Simple Mern Stack Project (for my students)
> MERN Stack deployment for Hotel listings application.


### Resources
[How to Use MERN Stack: A Complete Guide](https://www.mongodb.com/languages/mern-stack-tutorial)

***



## Installation
**Start server:**
```
cd mern/server
npm install
npm start
```

**Start Web server**
```
cd mern/client
npm install
npm start
```
